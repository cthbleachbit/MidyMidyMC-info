cth 的强迫症式铁路规范
======

共有两个部分：高架（包括地面和穿越地形的铁路）和地下铁路，又分高度不变的直线铁路、上坡、下坡、转弯和站台五种子类别。

在此之前

材料：
------
* 可以制作半砖和台阶的材料：木板、石英块、圆石、石砖、红砖、沙石、红沙石、地狱砖
* 单线使用的照明方块：火把、萤石、海晶灯
* 双线使用的照明放块：萤石、海晶灯
* 隔离用方块：玻璃（染色玻璃）、玻璃片（染色玻璃片）、木质栅栏、石墙
* 轨道及电路：动力铁轨、铁轨、拉杆、红石火把、红石粉、红石中继器、粘性活塞

图例：
------
* `-` 动力铁轨
* `~` 隔离
* `,` 拉杆
* `^` 照明
* `U` `D` 上下
* `L` `R` 左右
* `A` `B` `H` `T` 空气、方块、半砖、台阶
* `R` 轨道
